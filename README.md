## File Structure
- 책의 장을 기준으로 폴더를 분리합니다.
- 동일한 장 내에서 변경되는 코드들은 `Commit`으로 기록합니다.

<br/>

## Commit Convention
- `Example p.${page-number}-${example-number}` 의 형태로 작성합니다.  
- `example-number` 는 페이지에 여러개의 예제가 있을 때 추가합니다.
> Example p.43  
> Example p.50-1
